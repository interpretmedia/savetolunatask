## Save to LunaTask Chrome Extension
A lightweight Chrome extension that lets you send selected text or the current page link from any website or web app to LunaTask as either a note or a task. It supports appending repeated captures into the same note, works cleanly inside modern web apps using Shadow DOM isolation, and adapts to light or dark mode automatically. 
P.S. The project was vibe coded with Claude Code.

### What it does
1. Right-click selected text and save it to LunaTask as a note or task.

2. Click the extension icon in the Chrome toolbar to save the current page link as a note or task.

3. Append repeated saves into the same note using local cached content plus LunaTask's note update endpoint.
