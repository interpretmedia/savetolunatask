API Access Token
How to get your token: In LunaTask, go to Settings → Access Tokens → create a new token → copy it here. This token lets the extension create notes and tasks on your behalf. Keep it private — anyone with this token can access your LunaTask data.
ℹ How appending works LunaTask's API replaces note content on each write (E2E encrypted — server can't read/merge). This extension caches your note content locally and appends new captures to it before sending. Important: edits you make to the note inside LunaTask won't be visible here — use a dedicated "capture" note for this extension.
Your Capture Notes
Finding a Note's UUID: Open the note in LunaTask → click ⋯ → Share or Copy link → extract the UUID (format: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx) from the URL. Alternatively, check the note's Settings page.
Areas of Life
Why this is required: LunaTask's API requires area_id to create any task — there's no default. You must associate every task with an Area.

Finding an Area's UUID: In LunaTask, open an Area → click ⋯ → Settings → copy the UUID shown in the URL or settings page.
Goals
Goals are optional when saving tasks. If selected, the task will be linked to that goal inside LunaTask.

Finding a Goal's UUID: In LunaTask, open a Goal → click ⋯ → Settings → copy the UUID from the URL.
