How to get your token: In LunaTask → Settings → Access Tokens → create a token → paste it above. This lets the extension create notes and tasks on your behalf. Keep it private.
ℹ How append works LunaTask's API replaces note content on every write (E2E encrypted — the server can't merge). This extension caches your note content locally and appends new captures before each save. Use a dedicated capture note — hand-edits in LunaTask will be overwritten.
Capture Notes
Finding a Note UUID: Open the note → click ⋯ → Share or Copy link. Extract the UUID from the URL — format: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx.
Areas of Life
Why required: LunaTask's API mandates area_id for every task — there's no default.

Finding an Area UUID: Open the area → ⋯ → Settings → copy the UUID from the page URL.
Goals
Goals are optional — only shown when you save a task. Linking to a goal associates the task inside LunaTask.

Finding a Goal UUID: Open the goal → ⋯ → Settings → copy the UUID from the URL.
